# NeurIPS 2026 Communication Paper Scaffold

This directory holds the Phase 4 paper package for the DSC-EPGG communication paper.

## Inputs
- Main learned/no-comm 150k suite:
  - `outputs/eval/phase3_annealed_ext150k_5seeds/`
- Same-checkpoint continuation suite:
  - `outputs/eval/phase3_sameckpt_continuation_5seeds/`
- Statistics:
  - `outputs/eval/phase3_stats/`
- Interoperability summary:
  - `outputs/eval/phase3_interoperability/`

## Files
- `main.tex`: manuscript skeleton
- `refs.bib`: bibliography scaffold
- `FIGURE_TABLE_MANIFEST.md`: source-of-truth mapping from figures/tables to CSV outputs
- `RESULTS_MEMO.md`: compact writing memo with claims and source paths
